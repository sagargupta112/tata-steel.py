# Tata Steel HackerEarth Challenge

## Project Overview
This project is developed for the Tata Steel HackerEarth Machine Learning Challenge.

The application predicts output values using Machine Learning algorithms and provides a clean web-based user interface.

---

## Features

- Machine Learning Prediction System
- Flask Web Application
- User-Friendly Interface
- Clean Project Structure
- Automatic Submission File Generation

---
## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS
